집 안의 문제 해결하기

집에 문제가 생겼을 때 관리자나 집주인에게 요청하는 방법을 배워봅시다.

어휘 문법 문장 만들기 글 구성

1. 어휘 및 표현

물이 안 나오다

Water isn't coming out

하수구가 막히다

The drain is clogged

전등이 나가다

The light is out

보일러가 고장 나다

The boiler is broken

악취가 나다

There's a bad smell

수리/조치/해결

Repair / Action / Solution

카드를 클릭하여 뜻을 확인하세요.

2. 주요 문법

-(으)ㄴ 지 ...이/가 되다/넘다/지나다

어떤 일이 시작되고 시간이 얼마나 흘렀는지 나타낼 때 사용합니다.

예) 화장실 불이 나간 지 3일이 되었습니다.

-아/어 주시기 바랍니다

상대방에게 정중하게 무언가를 요청할 때 사용합니다.

예) 가능한 한 빨리 수리해 주시기 바랍니다.

-는 것 같다

확실하지 않은 사실에 대해 추측하여 말할 때 사용합니다.

예) 싱크대 배관에 문제가 생긴 것 같습니다.

3. 문장 만들기

배운 어휘와 문법을 사용해서 문장을 만들어 보세요.

문제 1: 이틀 전부터 부엌 싱크대 물이 잘 안내려갑니다. 집주인에게 이 상황을 설명하는 문장을 만드세요.

예시: 이틀 전부터 싱크대 물이 안 내려가기 시작했습니다. 싱크대가 막힌 지 이틀이 되었습니다.

문제 2: 화장실 변기가 고장 난 것 같습니다. 빠른 수리를 요청하는 문장을 만드세요.

예시: 화장실 변기가 고장 난 것 같습니다. 확인해 보시고 수리해 주시기 바랍니다.

4. 글 구성하기

아래 순서와 예시를 참고하여 집주인이나 관리자에게 보낼 글을 직접 작성해 보세요.

글의 순서

  1. 인사 및 자기소개: 몇 호에 사는지 밝힙니다.
  2. 문제 상황 설명: 언제부터, 어디에, 어떤 문제가 생겼는지 구체적으로 작성합니다.
  3. 요청 사항: 원하는 해결 방법과 처리 기한을 정중하게 요청합니다.
  4. 끝인사: 연락을 기다린다는 내용으로 마무리합니다.
